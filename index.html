Sobre o Curso de Desenvolvimento de Sistemas

O curso técnico em Desenvolvimento de Sistemas tem sido uma jor.nada incrível. Nele, aprendemos a criar soluções tecnológicas, desde a programação de softwares até o desenvolvimento de sites e aplicativos. É uma área que nos desafia a pensar de forma lógica e criativa para resolver problemas do dia a dia.

Quatro Temas Incríveis sobre Inteligência Artificial

Visão Computacional

A Visão Computacional permite que computadores "vejam" e interpretem imagens e vídeos, sendo fundamental em carros autônomos e reconhecimento facial.

Processamento de Linguagem Natural (PLN)

O PLN faz com que máquinas entendam a linguagem humana. É a tecnologia por trás de assistentes de voz como a Siri e o Google Assistant.

Aprendizado de Máquina (Machine Learning)

Com o Machine Learning, os computadores aprendem a partir de dados, sem serem explicitamente programados. É usado em recomendações de filmes e diagnósticos médicos.

Redes Neurais Artificiais

Inspiradas no cérebro humano, as Redes Neurais são a base para a criação de inteligências artificiais complexas, como as usadas para gerar imagens e textos.

Projeto Futurista para a SETEC_

- Tecnologia: Utilize Visão Computacional e Aprendizado de Máquina para treinar um modelo que identifique diferentes tipos de lixo (plástico, papel, metal, vidro). Funcionalidades: - Detecção de Imagens: Mostre a uma câmera uma imagem de lixo e o sistema identifica se é reciclável e qual o tipo. - Dashboard de Dados: Apresente um painel (dashboard) que mostre estatísticas como a quantidade de cada tipo de material classificado O projeto poderia ser o protótipo de um sistema de "lixeiras inteligentes" que, no futuro, seriam instaladas no colégio para monitorar o descarte de lixo e incentivar a reciclagem, gerando dados sobre os hábitos de consumo da comunidade escolar.
